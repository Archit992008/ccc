# CodeVenture (HTML Version)

CodeVenture is a gamified coding dashboard built with pure HTML & CSS.  
This version is a static landing page that shows a sidebar, user profile, progress, and coding worlds.

## Live Demo

Visit the live site here:  
https://<your-username>.github.io/<repo-name>/

## Features

- Sidebar with navigation and user profile  
- “Next Challenge” hero section  
- Progress card showing completed challenges & XP  
- Worlds section with individual challenge cards (difficulty + status)

## Tech

- HTML5 for layout and structure  
- CSS for dark theme and responsive design
